# Katkıda Bulunma Rehberi

Bu repoya katkıda bulunmak istiyorsanız aşağıdaki adımları izleyin.

## Hata Bildirimi

Bir hata bulduysanız lütfen [Issues](../../issues) sekmesinden şunları belirterek bildirin:
- Hatanın açık tanımı
- Yeniden üretme adımları
- Beklenen ve gerçekleşen davranış
- Python ve paket sürümleri (`pip freeze` çıktısı)

## Özellik Önerisi

Yeni bir özellik önermek için Issues sekmesinden `enhancement` etiketi ile açın.

## Pull Request

1. Repoyu fork edin
2. Yeni bir branch oluşturun: `git checkout -b ozellik/aciklama`
3. Değişikliklerinizi commit edin: `git commit -m 'feat: kısa açıklama'`
4. Branch'i push edin: `git push origin ozellik/aciklama`
5. Pull Request açın

## Kod Stili

- PEP 8 uyumlu Python kodu
- Fonksiyonlar için NumPy formatında docstring
- Türkçe yorum satırları tercih edilir
